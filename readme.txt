git命令行
  初期化:git init
  添加所有文件：git add .
  提交：git commit -m "git init commit"   后边是提交时写的注释
  关联github：git remote add origin https://github.com/Maraxianguo/locallibrary.git
  把上述提交进 master 分支 ：git push -u origin master

  第二次及以后提交更新都是 git add . 和 git commit
  然后 git push
  查看状态命令 git status
  差分查看 git diff
